## 📁 Project Access
To try it is very simple:<br>
-Clone the repository
```bash
$ git clone https://github.com/julianofrodrigues/e-Sport-Mobile
```
-Download the libraries
```bash
$ expo install
```
-Start the Project
```bash
$ expo start
```
